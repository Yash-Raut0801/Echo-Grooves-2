# Echoo

This is a full-stack web application with a **Node.js backend** and a **vanilla JavaScript frontend**.

## 📁 Project Structure

```
project-root/
│
├── backend/
│   ├── node_modules/
│   ├── package.json
│   ├── package-lock.json
│   └── server.js
│
├── frontend/
│   └── public/
│       ├── index.html
│       ├── app.js
│       └── style.css
│
├── node_modules/
├── package.json
├── package-lock.json
└── .gitignore
```

---

## 🚀 Features

* Backend built with Node.js
* Frontend using HTML, CSS, and JavaScript
* Simple project separation (frontend & backend)

---

## 🛠️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

#### Backend:

```bash
cd backend
npm install
```

#### Root (if applicable):

```bash
cd ..
npm install
```

---

## ▶️ Running the Project

### Start Backend Server

```bash
cd backend
node server.js
```

Or (if using nodemon):

```bash
npx nodemon server.js
```

### Open Frontend

* Navigate to:

```
frontend/public/index.html
```

* Open it in your browser
